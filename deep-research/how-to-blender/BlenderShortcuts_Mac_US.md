## Blender ショートカット — Mac × US キーボード (Mac 備え付け)

> ベース: [BlenderShortcuts.md](./BlenderShortcuts.md) (Blender Guru 版)  
> キーボード: MacBook 内蔵 US キーボード — US 配列、テンキーなし

---

## 初回セットアップ (必須)

| 設定 | 場所 |
| :---- | :---- |
| **Emulate Numpad** を ON | Edit > Preferences > Input |
| **Emulate 3 Button Mouse** を ON | Edit > Preferences > Input *(トラックパッド使用時)* |
| **ファンクションキーを標準 F キーとして使用** を ON | システム設定 > キーボード |

> ファンクションキー設定をしない場合、F1〜F12 はデフォルトで輝度・音量などのメディアキーになる。  
> 設定後は F12 → レンダリング、F11 → レンダリング再生 などが直接使えるようになる。  
> 設定しない場合は `fn + F12`、`fn + F11` で代替。

---

## 凡例

- **Ctrl** = ⌃ Control (Blender on Mac では ⌘ Command **ではない**)
- **Alt** = ⌥ Option
- **Delete** (前方削除) = `fn + ⌫` *(Mac 内蔵キーボードに独立した Delete キーなし)*
- **テンキー数字** → Emulate Numpad 有効時は**数字行**で代替 (後述)

---

## macOS との競合キー

| ショートカット | Blender の機能 | macOS の動作 | 対処 |
| :---- | :---- | :---- | :---- |
| `Ctrl + Space` | エリア最大化 | 入力ソース切替 / Spotlight 補完 | システム設定 > キーボード > ショートカットで競合を解除 |
| `F11` | レンダリング再生 | Mission Control / アプリウィンドウ | ファンクションキーを標準設定にするか `fn + F11` |
| `F12` | レンダリング | (通常競合なし) | ファンクションキーを標準設定にすれば OK |

---

## Delete キーについて

Mac 内蔵キーボードには独立した Delete (前方削除) キーがない。

| 用途 | キー |
| :---- | :---- |
| Blender でオブジェクト削除 | **`X`** *(推奨 — 確認ダイアログあり)* |
| 前方削除が必要な場合 | `fn + ⌫ (Backspace)` |

---

## ビュー (3D ビューポート) — テンキー → 数字行

Mac 内蔵キーボードはテンキーなしのため、Emulate Numpad 有効時は数字行で代替。

|  | / アイソレート |  |
| :---: | :---: | :---: |
| **7 天面 (Top)** | 8 上へ回転 | 9 反対 |
| 4 左へ回転 | **5 透視/平行切替** | 6 右へ回転 |
| **1 正面 (Front)** | 2 下へ回転 | **3 側面 (Side)** |
| **0 カメラ** |  | **. フォーカス** ※1 |

> ※1 テンキー `.` は Emulate Numpad 時に**未割当て**の場合あり。  
> Preferences > Keymap > 3D View > View > Align View > **View Selected** を任意キーに設定。  
> テンキー `/` (アイソレート) も同様に未割当ての場合あり。

| 機能 | キー |
| :---- | :---- |
| View Pie Menu | `` ` `` キー (1 の左、~ と共有) |
| 高速ビュー切替 | Alt + MMB (ドラッグ) |
| 全オブジェクト表示 | Home |
| 範囲にフォーカス | Shift + B |
