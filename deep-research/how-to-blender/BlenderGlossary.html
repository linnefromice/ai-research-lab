Blender 用語集 / 補足資料

ショートカット集 (BlenderShortcuts_Mac_*) で登場する概念・機能名の早見表 ／ 「このキーを押すと何が起きるか」を理解するための補足
🖱 マウスボタン略記
略記正式名説明
LMBLeft Mouse Button左クリック。選択・確定。トラックパッドでは通常クリック
RMBRight Mouse Button右クリック。コンテキストメニュー・操作キャンセル。トラックパッドでは 2 本指クリック
MMBMiddle Mouse Buttonホイールクリック。視点操作 (Orbit/Pan) に多用。トラックパッドでは要設定 (Emulate 3 Button Mouse)
Scrollマウスホイールズームイン/アウト、フレームスクロールなど
🖥 インターフェース
3D ビューポート
Blender のメイン作業エリア。3D 空間でオブジェクトを表示・操作する場所。画面の大半を占める
N パネルサイドバー
ビューポート右端のパネル。位置/回転/スケールの数値入力、アドオン設定など。N でトグル
T パネルツールバー
ビューポート左端。アクティブなツール (移動・押し出しなど) のアイコンが並ぶ。T でトグル
アウトライナーOutliner
画面右上のシーン構造ツリー。オブジェクト・コレクションの親子関係を管理する
プロパティエディター
画面右下。オブジェクト/マテリアル/物理シミュレーションのプロパティを編集。Shift+F7
ヘッダーHeader
各ウィンドウ上部の細いバー。モード切替・表示オプションのメニューがある
クアッドビューQuad View
ビューポートを正面/側面/天面/透視の 4 分割で表示。正確な位置合わせに便利。Ctrl+Alt+Q
3D カーソル
3D 空間に置ける特殊な点。原点・ピボット・スナップターゲットに使用。Shift+C で中心へ
🔄 モード (Mode)
モード説明切替
オブジェクトモードシーン全体のオブジェクトを選択/移動/配置。デフォルトモードTab
編集モード選択オブジェクトの頂点/エッジ/フェースを直接編集するTab
スカルプトモード粘土を彫るように有機的な形状を作るCtrl+Tab パイ
ポーズモードアーマチュアのボーンを動かしてポーズ/アニメーションを作るCtrl+Tab パイ
ウェイトペイントメッシュの各頂点にボーンの影響度 (ウェイト) を塗るCtrl+Tab パイ
透視投影Perspective
遠近感あり・現実と同じ見え方。デフォルト。5 (Emulate Numpad) でトグル
平行投影Orthographic
遠近感なし・奥行き方向に歪まない。正確な形状確認に便利。5 でトグル
👁 ビュー操作
オービットOrbit
視点をシーン中心周りに回転。MMB ドラッグ
パンPan
視点を上下左右に平行移動。Shift+MMB
フライモードFly Mode
FPS ゲームのように視点を移動。起動後 WASD + マウスで操作。Shift+`
アイソレートIsolate
選択オブジェクトだけを表示し他を非表示に。集中して作業したいときに。Numpad /
X-Ray
メッシュを半透明化して裏側の要素も選択可能にする。Edit Mode での選択作業に必須。Alt+Z
フォーカスFocus / View Selected
選択オブジェクトを画面中央に収めてズーム。行方不明のオブジェクト探しに。Numpad .
↔ 変換操作 (Transform)
移動Grab / Move
G → X/Y/Z で軸限定。数値直接入力も可 (例: G Z 2 Enter で Z 軸方向に 2 移動)
スケールScale
S → 軸指定可。S 2 Enter で 2 倍に拡大
回転Rotate
R → 軸指定可。R Z 45 Enter で Z 軸 45° 回転
ローカル軸
オブジェクト自身の座標軸。R → X,X のように軸キーを 2 回押すとローカル軸での操作になる
ピボットポイントPivot Point
回転/スケールの中心点。個別オリジン (各オブジェクトの原点) / 3D カーソル / アクティブ要素 など。. パイで切替
変換オリエンテーション
X/Y/Z 軸の向きの基準。Global (ワールド座標) / Local (オブジェクト固有) / Normal (フェースの法線方向) など。, パイで切替
スナッピングSnapping
移動/回転/スケールを格子点・頂点・面などに吸着。Shift+Tab でトグル。Shift+S パイで座標コピーも可
プロポーショナル編集
選択頂点を動かすと周囲の頂点も追従して変形。スムーズな地形・有機体の変形に。O でトグル。Scroll でフォールオフ幅変更
適用Apply
位置/回転/スケールを「初期値 (0/0/1)」として確定。モディファイアーの確定にも使用。Ctrl+A
クリアClear
変換を初期値に戻す。位置クリア Alt+G / 回転クリア Alt+R / スケールクリア Alt+S
📦 オブジェクト管理
複製Duplicate
独立したコピーを作成。Shift+D
リンク複製Linked Duplicate
メッシュデータを共有するコピー。片方の形状を変えると両方変わる。Alt+D
結合Join
複数オブジェクトを 1 つに合体させる。Ctrl+J
親子関係Parent / Child
親を動かすと子も追従。キャラの持ち物や車輪などに。Ctrl+P で設定 / Alt+P で解除
コレクションCollection
オブジェクトをグループ管理する単位。アウトライナーのフォルダ相当。M でコレクション移動
サブディビジョン
メッシュを細分化して滑らかにするモディファイアー。数値が高いほど細かく滑らか。Ctrl+数字
ミラーMirror
X/Y/Z 軸を指定してオブジェクトを鏡像変換。Ctrl+M → 軸指定
🔷 メッシュ要素 (Edit Mode)
名称英語説明選択モード
頂点Vertex / Verticesメッシュの点。すべての形状の基本単位1
エッジEdge2 頂点を結ぶ線2
フェースFace3 頂点以上で囲まれた面。Tri (三角) / Quad (四角) / Ngon (多角形)3
法線Normalフェースの「表」の向きを示すベクトル。逆向きだとレンダリングで黒く見える。Shift+N で再計算—
ループ選択Loop Selection
エッジ/フェースのループを一括選択。Alt+LMB。四角形メッシュ (クワッドポリ) が前提
リング選択Ring Selection
エッジループに垂直なエッジ列を選択。Ctrl+Alt+RMB
リンク選択Linked Selection
接続されているメッシュ全体を選択。Ctrl+L (全体) / L (カーソル下)
🔧 モデリングツール (Edit Mode)
押し出しExtrude
選択頂点/エッジ/フェースを引き出して新しいジオメトリを追加。最頻出操作。E
インセットInset
フェースを内側に縮小した新しいフェースを作る。窓枠・穴の作成に。I
ベベルBevel
エッジを面取り (斜め切り) して滑らかにする。Ctrl+B。Scroll でセグメント数調整
ループカットLoop Cut
メッシュを一周切断して新しいエッジループを追加。Ctrl+R。Scroll で本数変更
スライドSlide
頂点/エッジを隣接エッジに沿って移動。形状を維持したまま位置調整。G,G
ナイフKnife
自由な形でメッシュを切断する。K で起動、Enter で確定
フィルFill Face
選択エッジ/頂点を囲んでフェースを作成。F
マージMerge
複数頂点を 1 点にまとめる。M でどこへマージするか選択 (中心/カーソル/最初/最後)
リップRip
エッジ/頂点を裂いて隙間を作る。V。フィルなしの穴あけ
シアーShear
選択要素を平行四辺形状に変形。Ctrl+Shift+Alt+S
ベンドBend
選択要素を弧状に曲げる。Shift+W
エッジクリースEdge Crease
サブディビジョン時のエッジの鋭さを制御。値が高いほど鋭くなる。Shift+E
〰 カーブ / UV テクスチャリング
ハンドルHandle
ベジェカーブの接線方向を制御する補助点。Auto / Vector (直線) / Aligned (対称) / Free (自由)。V でタイプ変更
ティルトTilt
カーブの「ねじれ」角度。カーブを押し出したときの面の向きを制御。Ctrl+T
アンラップUnwrap
3D メッシュを展開して 2D の UV マップを作る。紙箱を展開図にするイメージ。U
シームSeam
UV 展開時の「切れ目」になるエッジ。シームが UV アイランドの境界になる。Ctrl+E でマーク
UV アイランド
UV マップ上でシームで区切られた島状の領域。L でアイランド選択
ピンPin
UV 頂点を固定して再アンラップ時に動かないようにする。P
💡 シェーディングモード — Z パイで切替
Wireframe
ワイヤーフレームのみ。内部構造・裏側の確認に。X-Ray 的な用途
Solid
デフォルト。マテリアルなしの面表示。モデリング作業の基本
Material Preview
マテリアルを反映したプレビュー。HDR ライト環境で表示。テクスチャ確認に
Rendered
レンダーエンジンでのリアルタイムプレビュー。EEVEE なら高速で確認可能
🔗 ノード (Node Editor)
ノードNode
マテリアル・コンポジット・ジオメトリを視覚的なブロックとワイヤーで構成するシステム
ソケットSocket
ノードの入出力端子。黄: Color / 灰: Float / 紫: Vector など色で種類を識別
フレームFrame
関連ノードをグループ化して視覚的に整理する入れ物。Ctrl+J
ミュートMute
ノードを削除せずバイパス (無効化) する。M。効果の比較に便利
コンポジターCompositor
レンダリング結果に後処理 (グロー/色補正/合成など) を加えるノードエディター
バックドロップBackdrop
コンポジターでのプレビュー画像。Alt+MMB で移動、V でズーム
🗿 スカルプト (Sculpt Mode)
ブラシBrush
スカルプトの作業ツール。サイズ F、強度 Shift+F、種類 Shift+Space で切替。Ctrl ホールドで反転効果
マスクMask
スカルプトの影響を受けないよう頂点を保護。M で塗り、Ctrl+I で反転、Alt+M でクリア
フェースセットFace Set
フェースにグループ ID を割り当て。選択範囲の限定・非表示・スカルプト範囲の制御に使う。Alt+W パイ
🎞 アニメーション
キーフレームKeyframe
アニメーションの「要点」フレーム。2 つのキーフレームの間は自動補間される。I で現在フレームに追加
タイムラインTimeline
フレームと再生コントロールのウィンドウ。キーフレームの大まかな配置確認に使う
ドープシートDopesheet
全オブジェクト/ボーンのキーフレームを一覧表示する。タイムライン上の表示を詳細化したもの。Ctrl+Tab
グラフエディターGraph Editor
キーフレームの補間カーブ (F-Curve) を編集。動きのイージングを細かく調整できる
F-Curve
フレームと値の関係を表すアニメーションカーブ。グラフエディターで視覚的に編集する
補間Interpolation
2 つのキーフレーム間の値の変化の仕方。Bezier (なめらか)・Linear (等速)・Constant (瞬間切替)。T で変更
イージングEasing
動きの加速/減速の制御。Ease In (入り遅い)・Ease Out (出遅い)。F-Curve のハンドルで調整
外挿Extrapolation
アニメーション範囲外 (最初・最後のキーの外側) での振る舞い。Shift+E で設定
マーカーMarker
タイムライン上の名前付き目印。カメラ切替・シーン区切りなどに使用。M
プレビュー範囲Preview Range
再生範囲を一部に限定する。特定のシーンだけを繰り返し確認したいときに。P→ドラッグ
🦴 リギング / アーマチュア
アーマチュアArmature
キャラクターを動かすためのボーン (骨格) の集合体。独立したオブジェクトとして存在する
ボーンBone
アーマチュアを構成する 1 本の骨。Head (根元) と Tail (先端) を持ち、階層構造を作る
ロールRoll
ボーンの軸回りの向き。IK の動作方向やウェイトペイントに影響。Ctrl+R で調整、Shift+N で再計算
IKInverse Kinematics
末端ボーン (例: 手) を動かすと親ボーン (腕) が自動追従。リアルな腕/足の動きに必須。Shift+I で追加
FKForward Kinematics
親ボーンから順に回転を設定して動かす通常の方式。繊細なポーズ制御に
コンストレイントConstraint
ボーン/オブジェクトの動きに制限・追従・コピーなどのルールを追加する機能。Ctrl+Shift+C
ブレークダウンBreakdown
2 つのポーズキーフレームの「中間」を示す特殊フレーム。Push/Relax でどちら寄りかを調整
ポーズライブラリPose Library
よく使うポーズを名前付きで保存しておく場所。Shift+L で追加
📸 レンダリング
レンダリングRendering
3D シーンを 2D 画像/動画に計算・出力する処理。静止画: F12、アニメーション: Ctrl+F12
Cycles
光の物理シミュレーションによる高品質レンダーエンジン。時間はかかるが写実的な出力が可能
EEVEE
リアルタイムレンダーエンジン。高速だがライティングは近似。アニメーション制作によく使われる
レンダリング範囲Render Region
ビューポートの一部だけをレンダリング。テスト時間の短縮に。Ctrl+B で設定、Ctrl+Alt+B でクリア
レンダースロット
レンダリング結果を最大 8 枚まで保持できるバッファ。J / Alt+J で切替。設定比較に便利

キーを押すと円形に選択肢が広がるメニュー。マウスを目的の方向へ動かして選ぶ。素早い操作に特化。
