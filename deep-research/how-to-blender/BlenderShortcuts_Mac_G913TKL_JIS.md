## Blender ショートカット — Mac × JIS キーボード (Logicool G913 TKL)

> ベース: [BlenderShortcuts.md](./BlenderShortcuts.md) (Blender Guru 版)  
> キーボード: Logicool G913 TKL JP — JIS 配列、テンキーなし

---

## 初回セットアップ (必須)

| 設定 | 場所 |
| :---- | :---- |
| **Emulate Numpad** を ON | Edit > Preferences > Input |
| *(トラックパッド使用時のみ)* **Emulate 3 Button Mouse** を ON | Edit > Preferences > Input |
| ファンクションキーを F キーとして動作させる | G HUB > キーボード設定 で確認 |

---

## 凡例

- **Ctrl** = ⌃ Control (Blender on Mac では ⌘ Command **ではない**)
- **Alt** = ⌥ Option
- **テンキー数字** → Emulate Numpad 有効時は**数字行**で代替 (後述)

---

## macOS との競合キー

| ショートカット | Blender の機能 | macOS の動作 | 対処 |
| :---- | :---- | :---- | :---- |
| `Ctrl + Space` | エリア最大化 | IME 切替 (JIS 環境) | システム設定 > キーボード > ショートカットで競合を解除 |
| `F11` | レンダリング再生 | Mission Control | `fn + F11` を使うか macOS 側を変更 |

---

## JIS 配列固有の差異

| 原典の記号 | US キーボードでの入力 | G913 TKL JIS での入力 |
| :---- | :---- | :---- |
| `` ~ `` (View Pie Menu) | `` ` `` キー (1 の左) | **キー不在 → 要リマップ** ※1 |
| `Shift + ~` (Fly) | `Shift + `` ` `` ` | **要リマップ** ※1 |
| `[` / `]` | そのまま | `「` / `」` キー (位置が右にずれるが同一キーコード) |

> ※1 JIS では同位置が `半角/全角` (英数・かな切替) キー。  
> **リマップ手順**: Edit > Preferences > Keymap > 3D View > Header > **View Pie Menu**  
> `^` キー (0 と - の間) や任意のキーに変更する。Fly も同様に設定。

---

## ビュー (3D ビューポート) — テンキー → 数字行

G913 TKL はテンキーなしのため、Emulate Numpad 有効時は数字行で代替。

|  | / アイソレート |  |
| :---: | :---: | :---: |
| **7 天面 (Top)** | 8 上へ回転 | 9 反対 |
| 4 左へ回転 | **5 透視/平行切替** | 6 右へ回転 |
| **1 正面 (Front)** | 2 下へ回転 | **3 側面 (Side)** |
| **0 カメラ** |  | **. フォーカス** ※2 |

> ※2 テンキー `.` は Emulate Numpad 時に**未割当て**の場合あり。  
> Preferences > Keymap > 3D View > View > Align View > **View Selected** を任意キーに設定。  
> テンキー `/` (アイソレート) も同様に未割当ての場合あり。

| 機能 | キー |
| :---- | :---- |
| View Pie Menu | ~ → **要リマップ** (JIS では `` ` `` キーなし) |
| 高速ビュー切替 | Alt + MMB (ドラッグ) |
| 全オブジェクト表示 | Home |
| 範囲にフォーカス | Shift + B |
