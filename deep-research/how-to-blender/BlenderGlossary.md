# Blender 用語集 / 補足資料

> ショートカット集 (BlenderShortcuts_Mac_*) で登場する概念・機能名の早見表。
> 「このキーを押すと何が起きるか」をすぐ確認できるよう、実用ベースで記述。

---

## マウスボタン略記

| 略記 | 正式名 | 説明 |
| :-- | :-- | :-- |
| LMB | Left Mouse Button | 左クリック。選択・確定。トラックパッドでは通常クリック |
| RMB | Right Mouse Button | 右クリック。コンテキストメニュー表示・操作のキャンセル。トラックパッドでは 2 本指クリック |
| MMB | Middle Mouse Button | ホイールクリック。視点操作 (Orbit/Pan) に多用。トラックパッドでは Emulate 3 Button Mouse を設定 |
| Scroll | マウスホイール | ズームイン/アウト、フレームスクロールなど |

## インターフェース

| 名称 | 説明 |
| :-- | :-- |
| **3D ビューポート** | Blender のメイン作業エリア。3D 空間でオブジェクトを表示・操作する場所 |
| **N パネル** (サイドバー) | ビューポート右端のパネル。位置/回転/スケールの数値入力、アドオン設定など。`N` でトグル |
| **T パネル** (ツールバー) | ビューポート左端。アクティブなツールのアイコンが並ぶ。`T` でトグル |
| **アウトライナー** | 画面右上のシーン構造ツリー。オブジェクト・コレクションの親子関係を管理 |
| **プロパティエディター** | 画面右下。オブジェクト/マテリアル/物理シミュレーションのプロパティを編集 |
| **ヘッダー** | 各ウィンドウ上部の細いバー。モード切替・表示オプションのメニューがある |
| **クアッドビュー** | ビューポートを正面/側面/天面/透視の 4 分割で表示。`Ctrl + Alt + Q` |
| **3D カーソル** | 3D 空間に置ける特殊な点。原点・ピボット・スナップターゲットに使用。`Shift + C` で中心へ |

## モード (Mode)

| モード名 | 説明 | 切替 |
| :-- | :-- | :-- |
| **オブジェクトモード** | シーン全体のオブジェクトを選択/移動/配置する。デフォルトモード | `Tab` / `Ctrl + Tab` |
| **編集モード** (エディットモード) | 選択オブジェクトの頂点/エッジ/フェースを直接編集する | `Tab` |
| **スカルプトモード** | 粘土を彫るように有機的な形状を作る | `Ctrl + Tab` パイ |
| **ポーズモード** | アーマチュアのボーンを動かしてポーズ/アニメーションを作る | `Ctrl + Tab` パイ |
| **ウェイトペイントモード** | メッシュの各頂点にボーンの影響度 (ウェイト) を塗る | `Ctrl + Tab` パイ |

## 投影モード

| 名称 | 説明 |
| :-- | :-- |
| **透視投影** (Perspective) | 遠近感あり。現実と同じ見え方。デフォルト |
| **平行投影** (Orthographic) | 遠近感なし・奥行き方向に歪まない。正確な形状確認に便利 |

`Numpad 5`（Emulate Numpad 有効時は数字行の `5`）でトグル。

## ビュー操作

| 名称 | 説明 |
| :-- | :-- |
| **オービット** (Orbit) | 視点をシーン中心周りに回転。`MMB` ドラッグ |
| **パン** (Pan) | 視点を上下左右に平行移動。`Shift + MMB` |
| **フライモード** (Fly Mode) | FPS ゲームのように視点を移動。起動後 WASD + マウスで操作 |
| **アイソレート** (Isolate) | 選択オブジェクトだけを表示し他を非表示にする。`Numpad /` |
| **X-Ray** | メッシュを半透明化して裏側の要素も選択可能にする。`Alt + Z` |
| **フォーカス** (Focus) | 選択オブジェクトを画面中央に収めてズーム。`Numpad .` |

## 変換操作 (Transform)

| 名称 | 説明 |
| :-- | :-- |
| **移動** (Grab) | オブジェクト/要素を移動。`G` → X/Y/Z で軸限定、数値入力も可 |
| **スケール** (Scale) | 拡大縮小。`S` → 軸指定可 |
| **回転** (Rotate) | 回転。`R` → 軸指定可 |
| **ローカル軸** | オブジェクト自身の座標軸。`R → X,X` のように軸キーを 2 回でローカル軸操作 |
| **ピボットポイント** | 回転/スケールの中心点。個別オリジン/3D カーソル/アクティブ要素など。`.` パイで変更 |
| **変換オリエンテーション** | X/Y/Z 軸の向きの基準。Global (ワールド座標) / Local / Normal など。`,` パイで変更 |
| **プロポーショナル編集** | 選択頂点を動かすと周囲の頂点も追従。スムーズな変形に。`O` でトグル、Scroll でフォールオフ幅 |
| **スナッピング** | 操作を格子点/頂点/面などに吸着させる。`Shift + Tab` でトグル |
| **精密移動** | `Shift` ホールドで移動量が細かくなる |
| **増分移動** | `Ctrl` ホールドでグリッド単位にスナップ |
| **適用** (Apply) | 位置/回転/スケールを「初期値 (0/0/1)」として確定。変形が正規化される。`Ctrl + A` |
| **ミラー** (Mirror) | X/Y/Z 軸を指定してオブジェクトを鏡像変換。`Ctrl + M` |
| **クリア** (Clear) | 位置 (Alt+G) / 回転 (Alt+R) / スケール (Alt+S) を初期値に戻す |

## オブジェクト管理

| 名称 | 説明 |
| :-- | :-- |
| **複製** (Duplicate) | 独立したコピーを作成。`Shift + D` |
| **リンク複製** (Linked Duplicate) | データを共有するコピー。片方を編集すると両方変わる。`Alt + D` |
| **結合** (Join) | 複数オブジェクトを 1 つに合体。`Ctrl + J` |
| **分離** (Separate) | 選択メッシュを別オブジェクトとして切り出す。`P` |
| **親子関係** (Parent/Child) | 親を動かすと子も追従。`Ctrl + P` で設定 / `Alt + P` で解除 |
| **コレクション** | オブジェクトをグループ管理する単位。アウトライナーのフォルダ相当。`M` で移動 |
| **サブディビジョン** | メッシュを細分化して滑らかにするモディファイアー。`Ctrl + 数字` でレベル追加 |

## メッシュ要素

| 名称 | 説明 | 選択モード |
| :-- | :-- | :-- |
| **頂点** (Vertex) | メッシュの点。すべての形状の基本単位 | `1` |
| **エッジ** (Edge) | 2 頂点を結ぶ線 | `2` |
| **フェース** (Face) | 3 頂点以上で囲まれた面。Tri (三角) / Quad (四角) / Ngon (多角形) | `3` |
| **法線** (Normal) | フェースの「表」の向きを示すベクトル。逆向きだとレンダリングで黒く見える。`Shift + N` で再計算 | — |

## 選択

| 名称 | 説明 |
| :-- | :-- |
| **ループ選択** | エッジ/フェースのループを一括選択。`Alt + LMB`。四角形メッシュ (クワッドポリ) が前提 |
| **リング選択** | エッジループに垂直方向のエッジ列を選択。`Ctrl + Alt + RMB` |
| **リンク選択** | 接続されているメッシュ全体を選択。`Ctrl + L` (全体) / `L` (カーソル下) |
| **ラッソ選択** | 手書きの輪郭で囲んだ範囲を選択。`Ctrl + RMB` ドラッグ |
| **サークル選択** | ブラシ形状で塗って選択。`C` で開始、Scroll でサイズ変更、`Enter` で確定 |
| **ボックス選択** | 矩形で範囲選択。`B` |

## モデリングツール (Edit Mode)

| 名称 | 説明 |
| :-- | :-- |
| **押し出し** (Extrude) | 選択頂点/エッジ/フェースを引き出して新しいジオメトリを追加。Blender 最頻出操作。`E` |
| **インセット** (Inset) | フェースを内側に縮小した新しいフェースを作る。窓枠・穴の作成に。`I` |
| **ベベル** (Bevel) | エッジを面取りして滑らかにする。`Ctrl + B`。Scroll でセグメント数調整 |
| **頂点ベベル** | 頂点を面取り (頂点を多角形に広げる)。`Ctrl + Shift + B` |
| **ループカット** (Loop Cut) | メッシュを一周切断して新しいエッジループを追加。`Ctrl + R`。Scroll で本数変更 |
| **スライド** (Slide) | 頂点/エッジを隣接エッジに沿って移動。形状を維持したまま位置調整。`G, G` |
| **ナイフ** (Knife) | 自由な形でメッシュを切断。`K` で起動、`Enter` で確定 |
| **フィル** (Fill) | 選択エッジ/頂点を囲んでフェースを作成。`F` |
| **シアー** (Shear) | 選択要素を平行四辺形状に変形。`Ctrl + Shift + Alt + S` |
| **ベンド** (Bend) | 選択要素を弧状に曲げる。`Shift + W` |
| **リップ** (Rip) | エッジ/頂点を裂いて隙間を作る。`V` |
| **リップフィル** (Rip Fill) | リップした隙間をフェースで塞ぐ。`Alt + V` |
| **マージ** (Merge) | 複数頂点を 1 点にまとめる。`M` でどこへマージするか選択 |
| **分割** (Split) | 選択要素をメッシュから切り離す（同じオブジェクト内に残る）。`Y` |
| **エッジクリース** (Edge Crease) | サブディビジョン時のエッジの鋭さを制御。`Shift + E`。値が高いほど鋭い |

## カーブ (Curve)

| 名称 | 説明 |
| :-- | :-- |
| **コントロールポイント** | カーブの形状を定義する点。ベジェカーブではハンドルを持つ |
| **ハンドル** (Handle) | ベジェカーブの接線方向を制御する補助点。タイプ: Auto / Vector (直線) / Aligned (対称) / Free (自由) |
| **ティルト** (Tilt) | カーブの「ねじれ」角度。カーブを押し出したときの面の向きを制御。`Ctrl + T` |

---

## UV / テクスチャリング

| 名称 | 説明 |
| :-- | :-- |
| **UV** | テクスチャ画像の 2D 座標系 (U = 横, V = 縦)。3D メッシュの各頂点を 2D 座標に対応付けたもの |
| **アンラップ** (Unwrap) | 3D メッシュを展開して 2D の UV マップを作る。紙箱を展開図にするイメージ。`U` |
| **シーム** (Seam) | UV 展開時の「切れ目」になるエッジ。`Ctrl + E` でマーク。シームが UV アイランドの境界になる |
| **UV アイランド** | UV マップ上でシームで区切られた島状の領域 |
| **スティッチ** (Stitch) | 隣接する UV アイランドを縫い合わせて結合。`V` |
| **ピン** (Pin) | UV 頂点を固定して再アンラップ時に動かないようにする。`P` |

## シェーディングモード

ビューポートの表示方式。`Z` パイメニューで切替。

| モード | 説明 |
| :-- | :-- |
| **Wireframe** | ワイヤーフレームのみ。内部構造の確認に |
| **Solid** | デフォルト。マテリアルなしの面表示。モデリング作業の基本 |
| **Material Preview** | マテリアルを反映したプレビュー。HDR ライト環境で表示 |
| **Rendered** | レンダーエンジンでのリアルタイムプレビュー。EEVEE なら高速 |

## ノード (Node Editor)

| 名称 | 説明 |
| :-- | :-- |
| **ノード** (Node) | マテリアル・コンポジット・ジオメトリをブロック (ノード) とワイヤー (リンク) で構成するシステム |
| **ソケット** (Socket) | ノードの入出力端子。黄: Color / 灰: Float / 紫: Vector など色で識別 |
| **フレーム** (Frame) | 関連ノードをグループ化して視覚的に整理する入れ物。`Ctrl + J` |
| **ミュート** (Mute) | ノードを削除せずバイパス (無効化) する。`M` |
| **グループ** | 複数ノードをひとまとめにして再利用可能な単位に。`Ctrl + G` でグループ化 |
| **コンポジター** | レンダリング結果に後処理 (グロー/色補正/合成など) を加えるノードエディター |
| **バックドロップ** | コンポジターでのプレビュー画像表示。`Alt + MMB` で移動 |

## スカルプト

| 名称 | 説明 |
| :-- | :-- |
| **ブラシ** (Brush) | スカルプトの作業ツール。サイズ `F`、強度 `Shift + F`、種類 `Shift + Space` で切替 |
| **マスク** (Mask) | スカルプトの影響を受けないよう頂点を保護。`M` で塗り、`Ctrl + I` で反転、`Alt + M` でクリア |
| **フェースセット** (Face Set) | フェースにグループ ID を割り当て。選択範囲の限定や非表示に使う。`Alt + W` パイ |

---

## アニメーション

| 名称 | 説明 |
| :-- | :-- |
| **キーフレーム** (Keyframe) | アニメーションの「要点」フレーム。2 つのキーフレームの間は自動補間。`I` で追加 |
| **タイムライン** | フレームと再生コントロールを持つウィンドウ。キーフレームの大まかな配置確認 |
| **ドープシート** (Dopesheet) | 全オブジェクト/ボーンのキーフレームを一覧表示するエディター。`Ctrl + Tab` で切替 |
| **グラフエディター** | キーフレームの補間カーブ (F-Curve) を編集。動きのイージングを細かく調整 |
| **F-Curve** | フレームと値の関係を表すアニメーションカーブ。グラフエディターで編集 |
| **補間** (Interpolation) | 2 つのキーフレーム間の値の変化の仕方。Bezier (なめらか) / Linear (等速) / Constant (瞬間切替)。`T` |
| **ハンドル** (F-Curve Handle) | F-Curve のベジェキーフレームに付く補助点。イージングを細かく制御。`V` でタイプ変更 |
| **外挿** (Extrapolation) | アニメーション範囲外 (最初・最後の外側) での振る舞い。`Shift + E` |
| **マーカー** (Marker) | タイムライン上の名前付き目印。カメラ切替・シーン区切りなどに使う。`M` |
| **プレビュー範囲** (Preview Range) | 再生範囲を一部に限定する。`P → ドラッグ` で設定、`Alt + P` でクリア |
| **逆再生** (Reverse Play) | アニメーションを逆方向に再生。`Ctrl + Shift + Space` |

## リギング / アーマチュア

| 名称 | 説明 |
| :-- | :-- |
| **アーマチュア** (Armature) | キャラクターを動かすためのボーン (骨格) の集合体。独立したオブジェクト |
| **ボーン** (Bone) | アーマチュアを構成する 1 本の骨。Head (根元) と Tail (先端) を持ち、階層構造を作る |
| **ロール** (Roll) | ボーンの軸回りの向き。IK の動作方向やウェイトペイントに影響。`Ctrl + R` で調整 |
| **IK** (Inverse Kinematics / インバースキネマティクス) | 末端ボーン (例: 手) を動かすと親ボーン (腕) が自動追従する仕組み。`Shift + I` で追加 |
| **FK** (Forward Kinematics / フォワードキネマティクス) | 親ボーンから順に回転を設定して動かす通常の方式 |
| **コンストレイント** (Constraint) | ボーン/オブジェクトの動きに制限・追従・コピーなどのルールを追加。`Ctrl + Shift + C` |
| **ブレークダウン** (Breakdown) | 2 つのポーズキーフレームの「中間」を示すフレーム。Push/Relax でどちら寄りかを調整 |
| **ポーズライブラリ** (Pose Library) | よく使うポーズを名前付きで保存。`Shift + L` で追加 |
| **Propagate Pose** (ポーズの伝播) | 現在のポーズを後続のキーフレームにも適用。`Alt + P` |
| **ボーンの溶解** (Dissolve Bone) | ボーンを削除して前後のボーンを直接接続。`Ctrl + X` |

---

## レンダリング

| 名称 | 説明 |
| :-- | :-- |
| **レンダリング** | 3D シーンを 2D 画像/動画に計算・出力する処理。`F12` で静止画 |
| **Cycles** | 光の物理シミュレーションによる高品質レンダーエンジン。時間がかかるが写実的 |
| **EEVEE** | リアルタイムレンダーエンジン。高速だがライティングは近似 |
| **レンダリング範囲** (Render Region) | ビューポートの一部だけをレンダリング。テスト時間短縮に。`Ctrl + B` で設定 |
| **レンダースロット** | レンダリング結果を最大 8 枚まで保持できるバッファ。`J` / `Alt + J` で切替 |

---

## パイメニュー (Pie Menu)

キーを押すと円形に選択肢が広がるメニュー。マウスを目的の方向へ動かして選ぶ。素早い操作に特化。

| キー | パイメニューの内容 |
| :-- | :-- |
| `Z` | シェーディングモード切替 (Wireframe / Solid / Material Preview / Rendered) |
| `.` | ピボットポイント切替 (個別オリジン / 3D カーソル / アクティブ要素 / 原点 / 重心) |
| `Shift + S` | スナップ (カーソルを選択へ / 選択をカーソルへ / グリッドへ など) |
| `,` | 変換オリエンテーション切替 (Global / Local / Normal / Gimbal / View / Cursor) |
| `Ctrl + Tab` | モード切替 (Object / Edit / Sculpt / Pose など) |
| `A` | スカルプトモード時のマスクパイ |
| `Alt + W` | スカルプトモード時のフェースセットパイ |
